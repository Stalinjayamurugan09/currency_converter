**Currency Converter in Python**
This project provides a simple Currency Converter implemented in Python using the currency_converter library. It allows you to convert an amount from one currency to another using exchange rates provided by the European Central Bank (ECB).

**Features**
Convert between supported currencies (e.g., USD to EUR, INR to USD, etc.).
Automatically fetches the latest exchange rates.
Provides user-friendly input and output formatting.

**To install the required library:**
pip install CurrencyConverter

**Example Usage**
Enter the amount: 100
Enter the base currency (e.g., USD): USD
Enter the target currency (e.g., EUR): EUR

100 USD is equal to 92.31 EUR
